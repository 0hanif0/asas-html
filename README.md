# ASAS HTML (HyperText Markup Language)

## Tags
- `<html> .. </html>` html
- `<h1>, <h2>, <h3>, <h4>` heading
- `<p>` paragraph
- `<em>, <strong>, <u>` format
- `<blockquote>` block
- `<ol>` ordered list
- `<ul>` unordered list
- `<li>` list item
- `<img>` gambar
- `<div>, <span>` blok & bahagian
- `<nav>` navigasi
- `<header>` header
- `<footer>` footer
- `<article>` kandungan
- `<table>` table
- `<thead>` table head
- `<tbody>` table body
- `<tr>` row
- `<th>` table head cell
- `<td>` row cell
- `<form>` borang
- `<button>` butang
- `<input>` input
- `<label>` label
- `<textarea>` input teks besar
- `<select>` dropdown
- `<option>` pilihan dropdown
